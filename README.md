# Floral – Çiçek & Aranjman Dünyası

Bu proje kapsamında çiçekçi sektörüne yönelik responsive bir web sitesi geliştirilmiştir. Kullanıcı dostu tasarım, kurumsal kimlik ve ürün tanıtımı odaklı oluşturulmuştur.

## Özet

Bu projede **Floral – Çiçek & Aranjman Dünyası** markası oluşturularak,
HTML5, CSS3, JavaScript, Bootstrap ve Chart.js teknolojileriyle çok sayfalı bir web sitesi geliştirilmiştir.

Web sitesi şunları içermektedir:

- Ana Sayfa
- Ürünler & Ürün Detay
- Duyurular
- Birimler
- Misyon – Vizyon
- İletişim (Google Maps + Form)
- Chart.js ile İstatistik Sayfası

Ek olarak admin panel arayüzleri hazırlanmıştır.

## Şablon ve Özelleştirmeler

Proje, **Colorlib – Alazea HTML Template** üzerine inşa edilmiştir.  
Aşağıdaki özelleştirmeler yapılmıştır:

- Renk paleti yenilendi 
- Logo ve kurumsal marka tasarımı oluşturuldu
- Menü yapısı yeniden düzenlendi
- Fontlar:
  - Başlıklar → *Playfair Display*
  - Metinler → *Dosis*
- Tüm görseller değiştirildi
- Çiçek sektörüne uygun tema haline getirildi
